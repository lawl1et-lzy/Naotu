### 项目概览

```
---
 |---- config/                      数据库配置项            
 |---- controler/                   接口层        
 |---- db/                          模型层和连接池      
 |---- public/                      公共资源     
 |---- util/                        工具类       
 |---- package.json                 配置信息            
 |---- api.js                       接口     
 |---- app.js                       项目启动入口          
 |---- README.md                    说明文档            
```

### 环境要求

1. NodeJS 8.9+ 环境
2. MongoDB 4.0.6+

### 运行
> node app.js / npm run apiserver
